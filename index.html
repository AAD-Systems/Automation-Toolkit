🎯 Sobre o Projeto

O Python Security Toolkit é um laboratório modular para desenvolvimento de ferramentas de segurança em Python. Cada componente é construído com foco em código limpo, logging profissional e tratamento robusto de erros, servindo como material de estudo para conceitos fundamentais de segurança ofensiva e defensiva.

⚙️ Diferenciais Técnicos

Performance Otimizada

Scanner multi-thread com suporte a múltiplas threads simultâneas. Ideal para estudos de concorrência em Python.

Métricas em Tempo Real

Sistema integrado de métricas para análise de performance e registro completo de scans realizados.

Logging Profissional

Sistema de logs com cores no terminal, múltiplos níveis (DEBUG, INFO, WARNING, ERROR) e saída em arquivo.

Múltiplos Formatos

Exporte resultados em JSON, CSV ou TXT. Ideal para análise posterior e integração com outras ferramentas.

Tratamento de Erros

Validação robusta de entradas e tratamento de exceções com fallbacks inteligentes.

Código Documentado

Type hints, docstrings completas e exemplos de uso para cada função.
