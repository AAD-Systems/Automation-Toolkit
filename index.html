Python Security Toolkit

Ferramentas profissionais para automatização de segurança, reconhecimento de rede e desenvolvimento de exploits. Código limpo, modular e pronto para produção.

Clone o Repositório Ver Exemplos

🎯 Diferenciais Técnicos

Alta Performance

Scanner multi-thread com suporte a até 500 threads simultâneas. Escaneie 10.000 portas em menos de 2 segundos.

Métricas em Tempo Real

Sistema integrado de métricas com registro de performance, portas por segundo e histórico completo de scans.

Logging Profissional

Sistema de logs com cores no terminal, múltiplos níveis (DEBUG, INFO, WARNING, ERROR) e saída em arquivo.

Sistema de Plugins

Arquitetura extensível com Plugin Manager para adicionar novas funcionalidades sem modificar o core.

Múltiplos Formatos

Exporte resultados em JSON, CSV ou TXT. Integração fácil com outras ferramentas e pipelines de dados.

Tratamento de Erros

Validação robusta de entradas, tratamento de exceções e fallbacks inteligentes para ambientes adversos.

🛠️ Ferramentas Disponíveis

TCP Scanner Simples

Scanner TCP básico para identificação de portas abertas. Ideal para reconhecimento inicial e ambientes com restrições.

python base/tcp_scanner_simple.py 192.168.1.1 -p 80,443,8080 -v

Multi-Thread Scanner

Scanner de alta performance com suporte a múltiplas threads. Escaneie ranges completos em segundos.

python base/multi_thread_scanner.py 192.168.1.1 -p 1-10000 -T 200 -o scan.json

Sistema de Métricas

Monitore performance em tempo real com métricas detalhadas de cada scan realizado.

from base import metrics
metrics.display()
